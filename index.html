ASIN Extractor

Automatically extract and manage ASINs from VSE Fixer with marketplace detection, smart filtering, and one-click copying.

Features
What it does

Powerful ASIN extraction and management for VSE Fixer workflows.

🎯

Auto-Detection

Automatically opens when ASINs are detected on the page. No manual triggering needed.

🌍

Marketplace Support

Supports 30+ Amazon marketplaces with color-coded badges and direct PDP links.

⚡

Smart Extraction

Extracts up to 10 ASINs with marketplace IDs using advanced pattern matching.

📋

Quick Copy

Select multiple ASINs and copy them with a single click. Comma-separated format.

🔄

Manual Reload

Reload button in the header to re-extract ASINs if auto-detection fails.

🎨

Compact Design

Slim, draggable widget that doesn't obstruct your workflow. Fixed buttons at bottom.

Installation
Get Started

One-click installation with Tampermonkey.

  1. First, install Tampermonkey extension if you don't have it (click button below)
  2. Once Tampermonkey is installed, click the Install Script button
  3. Tampermonkey will open with the script - click Install to confirm
  4. Navigate to https://prod.vse-fixer.tools.amazon.dev/*
  5. The widget will automatically open when ASINs are detected!
🔧 Install Tampermonkey 📥 Install Script
Note: Install Tampermonkey first, then click the Install Script button. The script will automatically prompt for installation in Tampermonkey.
How It Works
Simple & Automatic

The widget works seamlessly in the background.

1️⃣

Page Load

Script activates when you open VSE Fixer and monitors for ASINs.

2️⃣

Detection

Polls every second for ASINs in the page content using regex patterns.

3️⃣

Auto-Open

Widget opens automatically when ASINs are found, showing results immediately.

4️⃣

Fetch Integration

Clicking any "Fetch" button triggers automatic re-extraction with loading animation.
