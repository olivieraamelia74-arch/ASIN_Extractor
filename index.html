FAQ
Frequently Asked Questions

Everything you need to know about ASIN Extractor.

What is ASIN Extractor? ▼

ASIN Extractor is a Tampermonkey userscript that automatically detects and extracts ASINs from VSE Fixer pages. It displays them in a compact, draggable widget with features like multi-select, copy to clipboard, and marketplace tagging.

How do I install it? ▼

First, install Tampermonkey browser extension. Then click the "Install Script" button above. Tampermonkey will open with the script - click "Install" to confirm. The script will automatically activate on VSE Fixer pages.

Which browsers are supported? ▼

Fully supported: Chrome, Firefox, Edge. Limited support: Safari (requires Tampermonkey or Userscripts extension). The script works best on Chromium-based browsers.

Does it work automatically? ▼

Yes! The script monitors the page every 500ms for ASINs. When ASINs are detected, the widget opens automatically. You can also manually reload using the 🔄 button in the widget header.

What do the colored badges mean? ▼

The colored badges indicate the marketplace origin of each ASIN. For example: US for United States, UK for United Kingdom, DE for Germany, etc. This helps you quickly identify which marketplace each ASIN belongs to.

Can I select multiple ASINs? ▼

Yes! Check the boxes next to ASINs you want to copy. Click "Copy Selected" to copy them as a comma-separated list. Use "Unselect All" to clear your selection.

What if ASINs aren't detected? ▼

Click the reload button (🔄) in the widget header to manually re-extract ASINs. If the issue persists, check that you're on a VSE Fixer page with ASINs present. The script uses regex pattern matching to find ASINs.

Is my data stored anywhere? ▼

No. The script runs entirely in your browser and doesn't send any data to external servers. All ASIN extraction and processing happens locally on your machine.

How do I report bugs or request features? ▼

Use the feedback form above or click the floating 💬 button in the bottom-right corner. You can also reach out to any team member listed in the "Built by" section.
