FAQ
Frequently Asked Questions

Everything you need to know about ASIN Extractor.

Does it work on all browsers? ▼

ASIN Extractor works best on Chrome, Firefox, and Edge with Tampermonkey installed. Safari support is limited due to Tampermonkey compatibility issues. We recommend using Chrome for the best experience.

Will it slow down my browser? ▼

No! The script is highly optimized and only runs on VSE Fixer pages (https://prod.vse-fixer.tools.amazon.dev/*). It uses efficient polling (every 500ms) and stops automatically after 30 seconds if no ASINs are found.

How do I update to the latest version? ▼

Tampermonkey automatically checks for updates! You can also manually update by clicking the Tampermonkey icon → Dashboard → Check for updates. Or simply reinstall the script using the "Install Script" button above.

What if ASINs aren't detected? ▼

Try these solutions: 1) Click the manual 🔍 ASIN button (top-right), 2) Click the 🔄 Reload button in the widget header, 3) Click any "Fetch" button on the VSE Fixer page to trigger re-extraction, 4) Refresh the page and wait for auto-detection.

Can I customize the widget position? ▼

Yes! The widget is draggable. Simply click and hold the header (where it says "✨ ASINs") and drag it to your preferred position. Your browser will remember the position for future sessions.

How many ASINs can it extract at once? ▼

The tool extracts up to 10 ASINs per extraction to maintain performance and readability. If you need more, you can reload after copying the first batch.

Installation
Get Started

Quick and easy installation in just a few clicks.

  1. Click the Install Script button below to begin installation
  2. If you don't have Tampermonkey installed, you'll need to install it first (use the secondary button)
  3. Once Tampermonkey is ready, click Install Script again
  4. Tampermonkey will open with the script - click Install to confirm
  5. Navigate to https://prod.vse-fixer.tools.amazon.dev/*
  6. The widget will automatically open when ASINs are detected!
📥 Install Script 🔧 Install Tampermonkey
Note: If you don't have Tampermonkey installed yet, click the "Install Tampermonkey" button first, then return to install the script. The script will automatically prompt for installation in Tampermonkey.
How It Works
Simple & Automatic
